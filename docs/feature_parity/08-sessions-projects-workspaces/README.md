# Feature 3.8 – Sessions, Projects & Workspaces（功能 3.8 – 工作階段、專案與工作區）

## Scope / 範圍
- Session files capturing open tabs, cursor/scroll positions  
  工作階段檔案需記錄開啟分頁、游標與捲動位置
- Project panel with file/folder management, filters, quick open  
  專案面板支援檔案/資料夾管理、過濾器與快速開啟
- Workspace abstraction for multi-project switching and cross-project search  
  工作區抽象層，便於多專案切換與跨專案搜尋
